Case study / ShopSense

Conversational shopping assistant that replaces browse-heavy flow.

ShopSense turns product search and comparison into a chat experience with intent recognition, sentiment analysis, and recommendation output.

Role AI + full-stack developer
Core shift Online shopping to chat-first UX
Focus Product discovery quality

Problem

Standard e-commerce browsing creates too much friction for users who need fast recommendations, comparisons, and review synthesis.

Architecture and stack

  • Next.js + TypeScript for the chat-first product interface.
  • MongoDB text indexing for semantic product retrieval.
  • Gemini integration for response generation and sentiment tasks.
  • Appwrite for authentication and user workflow support.
  • SendGrid for recommendation email handoff.

Key decisions

  • Classified user intent to route product, review, and comparison requests.
  • Used chat memory to preserve user context through the session.
  • Added sentiment synthesis so users can evaluate products quickly.
  • Integrated email output for deferred purchase decisions.

Outcome

  • Shifted product exploration from static catalog browsing to guided conversations.
  • Reduced cognitive overhead by giving single-thread recommendations.
  • Enabled faster compare-and-decide flow for shoppers.
