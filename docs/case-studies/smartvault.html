Case study / SmartVault

AI-powered file management system for faster file discovery.

SmartVault is a full-stack storage platform focused on secure access, searchable file metadata, and AI-assisted file management.

Role Full-stack engineer
Scale 1000+ users
Outcome 50% faster file search

Problem

Users were spending too much time searching manually through cloud folders and metadata. The workflow lacked smart organization and natural search.

Architecture and stack

  • Next.js 15 App Router for UI and server rendering.
  • TypeScript for type-safe API and component contracts.
  • Appwrite for auth, database, and storage services.
  • Google Generative AI integration for smart file workflows.
  • Tailwind and component primitives for responsive UX.

Key decisions

  • Used structured metadata to reduce search surface early.
  • Designed authentication and access controls as first-class features.
  • Prioritized fast retrieval paths over feature-heavy navigation.
  • Kept deployment and local setup reproducible for fast iteration.

Impact

  • Used by 1000+ users.
  • Reduced time to locate files by about 50%.
  • Improved usability through simple file browsing and retrieval.
